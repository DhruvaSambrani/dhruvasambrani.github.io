Home - Drosophila Data Collection

In order to simplify the collection of data of the Drosophila Crosses, the following website has been made. Each of the above tabs are for each specific cross. Each cross has its own form that needs to be filled up. You will be asked to sign in using your Google account to make sure there is only one response from each person.

    The following are the rules to fill up the form -
  • Incase no flies were seen of a particular phenotype, mark as 0.
  • In case a cross was set up, but not succussful, mark 0 for each phenotype.
  • In case a cross wasn't set up at all (generally in F2), DO NOT fill up the form for THAT cross at all. This is done because no progeny is different from no cross.

You can view the results of the whole batch as they update it here. You may prefer to download the spreadsheet as an Excel file and edit the same OR go to File → Make a Copy and create a COPY of the file onto your Drive. Make sure you analyse the latest data only.
